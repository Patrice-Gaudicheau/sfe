# Historical OpenAI Smoke Report Snapshots

This directory preserves previously tracked generated Markdown reports from
OpenAI selector, executor, and combined smoke runs.

These files are historical snapshots, not the canonical current project status.
They remain useful as audit trail for narrow benchmark runs, but newly generated
logs and report streams should stay under ignored local paths such as `logs/`
unless they are intentionally summarized in `docs/` or `reports/`.
