+# NoteKeeper
+
+NoteKeeper is a small browser-only notes app scaffold built as a static application. It is designed to run directly from local files without a server, build step, or external dependencies.
+
+## Run locally
+
+1. Open the `app/` folder.
+2. Double-click `index.html`, or open it directly in a modern browser.
+
+## What's included in this scaffold
+
+- NoteKeeper app name and header
+- Search input placeholder
+- Active and archive view controls
+- Note creation form with title, body, note type, and labels fields
+- Checklist area placeholder for later implementation
+- Sample note cards showing text and checklist layouts
+- Baseline responsive styling for phone, tablet, and desktop widths
+
+## Current status
+
+This task delivers the initial static scaffold only. The interface is intentionally non-persistent at this stage. Later tasks will add note creation, editing, deletion, localStorage persistence, search, archive handling, checklist behavior, and pinning.
+
+For a quick manual check, open `index.html`, review the layout, tab through the controls, resize the browser window, and confirm the app remains usable without horizontal scrolling at smaller widths.
