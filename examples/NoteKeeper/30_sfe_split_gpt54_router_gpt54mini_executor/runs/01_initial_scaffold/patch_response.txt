+# NoteKeeper
+
+NoteKeeper is a small browser-only notes app scaffold. It is designed to open directly by double-clicking `index.html` or by loading the file in a browser from the `app/` folder.
+
+## Run
+
+1. Open `index.html` in a modern browser.
+2. No server, build step, or external dependency is required.
+
+## Current scaffold
+
+- App shell with the NoteKeeper name
+- Search field and active/archive view controls
+- Note creation form with title, body, labels, and note type controls
+- Placeholder checklist editing area
+- Empty-state note card area
+- Baseline responsive styling
+
+## Manual review
+
+- Confirm the page opens directly without a server.
+- Check that the header, form, controls, and note list are visible.
+- Resize the browser to verify the layout stays readable on narrow and wide screens.
+- Verify the visible product name is NoteKeeper.
